# Pliki indeksowo-sekwencyjne

To repozytorium zawiera implementację systemu plików indeksowo-sekwencyjnych w języku C++, zaprojektowaną do efektywnego przechowywania i przetwarzania dużych zbiorów danych. Projekt koncentruje się na optymalizacji operacji dyskowych poprzez wykorzystanie przetwarzania blokowego oraz zaawansowanych mechanizmów buforowania.



Implementacja obejmuje wieloplikowe przechowywanie danych, co umożliwia rozłożenie rekordów na kilka plików. Dzięki zastosowaniu techniki odczytu blokowego oraz optymalizacji buforowania, system znacznie redukuje liczbę operacji dyskowych i przyspiesza dostęp do danych. Rozwiązanie to pozwala na lepsze wykorzystanie zasobów systemowych i poprawę wydajności przy pracy z dużymi zbiorami danych.


Osiągnięcia

-Znaczące ograniczenie operacji dyskowych oraz przyspieszenie przetwarzania rekordów.

-Pogłębienie wiedzy na temat zarządzania dużymi zbiorami danych, struktur danych i optymalizacji pamięci w C++.

-Stworzenie skalowalnego i wydajnego systemu przechowywania danych, który może być wykorzystywany w aplikacjach wymagających intensywnego przetwarzania danych.
